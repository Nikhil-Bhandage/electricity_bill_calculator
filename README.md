# ⚡ Electricity Bill Calculator

A full-stack web application for calculating electricity consumption and estimating monthly electricity bills based on appliance usage. Built using Node.js, Express.js, MySQL, and Chart.js.

## 🚀 Features

- Calculate monthly electricity consumption (kWh)
- Estimate electricity bills in real time
- Add, update, and remove appliances
- Configure electricity rates, fixed charges, and taxes
- Interactive energy consumption charts
- MySQL database integration
- Responsive and modern UI

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript
- Tailwind CSS
- Chart.js

### Backend
- Node.js
- Express.js

### Database
- MySQL
